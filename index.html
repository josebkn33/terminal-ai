Herramientas disponibles

Usá las mismas IAs que en el navegador, pero más rápido y sin salir de tu terminal.

🟢

OpenAI (ChatGPT)

Consultá GPT-4 y GPT-3.5 desde la terminal. Ideal para preguntas, código, traducciones y más.

pip install openai
🟣

Claude (Anthropic)

Accedé a Claude 3 desde tu terminal. Perfecto para análisis de texto, documentación y razonamiento.

npm install @anthropic-ai/sdk
🔵

Ollama (Local)

Ejecutá modelos open-source como Llama 3, Mistral y CodeLlama localmente, sin internet.

ollama run llama3.2
🟠

GitHub Copilot CLI

Traducí lenguaje natural a comandos de terminal. Escribí qué querés hacer y Copilot lo ejecuta.

gh copilot suggest
⚪

Groq

IAs ultrarrápidas con inferencia en hardware dedicado. Ideal para respuestas en tiempo real.

pip install groq
🟤

ShellGPT

Wrapper que conecta con múltiples IAs desde la terminal. Instalación única, uso inmediato.

pip install shell-gpt

Cómo empezar en 3 pasos

Sin cuentas complicadas. Sin burocracia. Solo terminal.

1

Elegí tu herramienta

Decidí si querés usar la nube (ChatGPT, Claude) o algo local (Ollama). Para empezar, Ollama es la opción más simple porque no necesita API keys.

2

Instalá el cliente

Cada herramienta tiene un comando de instalación simple. Por ejemplo: brew install ollama o pip install shell-gpt

3

Empezá a preguntar

Ya está. Escribí tu primer prompt en la terminal. sgpt "¿qué es un blockchain en términos simples?"

Comandos útiles

Ejemplos prácticos para arrancar de una.

sgpt "crea un readme" Generar un README.md para tu proyecto
sgpt --code "script en python que descargue imágenes" Generar código directamente
sgpt --shell "matar proceso en puerto 3000" Traducir lenguaje natural a comandos shell
ollama run llama3 Ejecutar Llama 3 localmente
gh copilot suggest "find large files" Copilot te dice el comando exacto
ia "resumí este artículo: ..." Resumir textos largos al instante
